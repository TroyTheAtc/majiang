# 启动与重启命令

本仓库为静态 H5 项目，本地仅需启动简易 HTTP 服务即可在浏览器或手机同 WiFi 访问。以下命令均在**项目根目录**执行。

---

## 一、启动命令

进入项目目录并启动本地服务（端口 8081）：

```bash
cd "/Users/troy/Desktop/个人材料/000-宝宝/MAJIANG" && python3 serve.py
```

- 本机访问：<http://localhost:8081>
- 手机访问：`http://<本机IP>:8081`（手机与电脑需在同一 WiFi）
- 停止服务：在终端按 **Ctrl+C**

---

## 二、重启命令

先停止当前服务（Ctrl+C），再执行启动命令：

```bash
cd "/Users/troy/Desktop/个人材料/000-宝宝/MAJIANG" && python3 serve.py
```

或使用一条命令先结束占用 8081 端口的进程再启动（需根据系统替换 `lsof`/`kill` 用法）：

```bash
cd "/Users/troy/Desktop/个人材料/000-宝宝/MAJIANG"
# 结束已在跑的 serve（若存在）
kill $(lsof -t -i:8081) 2>/dev/null; python3 serve.py
```

---

## 三、重启 worker 命令

本仓库**无独立 worker 进程**，仅有一个 HTTP 服务（`serve.py`）。若用 **PM2** 等方式把该服务当“worker”管理，可如下操作。

### 使用 PM2 时

启动（并命名为 `majiang`）：

```bash
cd "/Users/troy/Desktop/个人材料/000-宝宝/MAJIANG" && pm2 start serve.py --name majiang --interpreter python3
```

重启（等同于“重启 worker”）：

```bash
pm2 restart majiang
```

查看状态：

```bash
pm2 status majiang
```
