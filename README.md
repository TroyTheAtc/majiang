手机端 H5 输赢记录，数据存浏览器本地。**H5 稳定版 v1.6.4.2**，鸿蒙端开发以此版本为基线。

## 公网部署（GitHub Pages，免费）

要获得公网地址、在外面用手机也能打开，请按 **「部署指南-GitHubPages.md」** 里的步骤操作，用 GitHub Pages 免费发布即可。

## 启动命令

在项目目录下执行：

```bash
./start.sh
```

或：

```bash
bash start.sh
```

**重启服务**（先停掉占用的端口再启动）：

```bash
./restart.sh
```

或复制到终端（任意目录均可）：

```bash
cd "/Users/troy/Desktop/个人材料/000-宝宝/MAJIANG" && ./restart.sh
```

启动后终端会显示访问地址。

## 网址

| 设备 | 地址 |
|------|------|
| **本机浏览器** | http://localhost:8081 |
| **手机浏览器** | http://\<电脑局域网IP\>:8081 |

手机需与电脑连接**同一 WiFi**，将上面 `<电脑局域网IP>` 换成启动时终端里显示的 IP（如 `192.168.1.100`）。

示例：若电脑 IP 为 `192.168.1.100`，则手机在浏览器输入：

**http://192.168.1.100:8081**

## 手机打不开网页时

1. **确认同一 WiFi**  
   手机和电脑必须连同一个 WiFi（不能一个 WiFi 一个手机热点）。

2. **关掉电脑防火墙对 Python 的拦截**  
   - 打开 **系统设置 → 网络 → 防火墙**（或 **隐私与安全性 → 防火墙**）  
   - 若防火墙已开：点「选项」→ 列表里找到 **Python** 或 **python3**，设为「允许传入连接」；若没有，先跑一次 `./start.sh` 再回来看列表。

3. **确认电脑 IP 没变**  
   终端执行：`ipconfig getifaddr en0`  
   看到的就是本机 IP，手机应访问 `http://这个IP:8081`。

4. **先在本机用局域网地址试**  
   电脑浏览器打开 `http://192.168.x.x:8081`（把 x.x 换成你电脑 IP）。  
   若**电脑上这个地址也打不开**，说明服务没对局域网开放，请用下面的「重启」再试。  
   若**电脑能打开、手机打不开**，多半是路由器做了「终端隔离」或手机/电脑没在同一 WiFi。

5. **运行诊断**（在项目目录执行）：  
   `./check.sh`  
   会显示本机 IP 和端口占用，并提示下一步怎么查。

6. **排除路由器隔离**  
   可试：手机开热点，电脑连手机热点，再在手机浏览器访问 `http://<电脑显示的本机IP>:8081`。若这样能打开，就是家里 WiFi 路由器限制了设备互访。

---

关闭终端或按 `Ctrl+C` 可停止服务。
